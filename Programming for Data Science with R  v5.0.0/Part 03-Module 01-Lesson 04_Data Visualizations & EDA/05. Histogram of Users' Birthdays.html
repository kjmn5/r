How to read Histograms and use them in R (note this post uses the base graphics package of R to create histograms; we'll be using the ggplot2 graphics package in this course)




Run the following code in R to get other themes.

install.packages('ggthemes', dependencies = TRUE)
library(ggthemes)



The ggthemes package was developed by Jeffery Arnold. Check out examples of the themes on the github page.




Chris is using theme_minimal()with the font size set to 24, which is why his output might look slightly different than yours. You can set the same theme in R by running the following code, or you can set the theme to a choice of your own.


theme_set(theme_minimal(24))






Instead of using the qplot() function, you can also use the ggplot() function to create the histogram:

ggplot(aes(x = dob_day), data = pf) +
  geom_histogram(binwidth = 1) +
  scale_x_continuous(breaks = 1:31)

For the most part, plots can be created using either qplot or ggplot. The reason for using one library vs. another is largely based on the personal experiences of the person writing the code. Some people are more comfortable with qplot syntax, while other people prefer ggplot syntax. However, in most circumstances, you can create the same plot in either library. For more information on this, you can visit the StackOverflow post here.

When you register at a website, some sites have default settings that use the 1st day of the month under the birthday field. Some also use a default month of January.

In the case of Facebook, Jan and the 1st are the first drop-down items that appear on the site. It's important to think about such details and the data generating process when exploring data.
