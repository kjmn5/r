01. What is EDA?

Exploratory Data Analysis (EDA)

In this lesson, you'll learn how to create data visualizations and explore data in R. This is just a glipse into a field called exploratory data analysis, commonly referred to as EDA. EDA is defined below.

"Exploratory data analysis (EDA) is a term for certain kinds of initial analysis and findings done with data sets, usually early on in an analytical process. Some experts describe it as “taking a peek” at the data to understand more about what it represents and how to apply it. Exploratory data analysis is often a precursor to other kinds of work with statistics and data." - Techopedia

What's Ahead

You will be applying your R skills to analyze data similar to Facebook data you might actually see if you worked at Facebook.

The layout for this lesson will be that you:

  1. Watch a video of how to approach a problem or build a visualization, where you are strongly encouraged to code along.

  2. A notebook with solution code is provided following the video, as well as some additional nuggets of useful information. This code is available in case you get stuck, as it is strongly recommended that you write the code yourself.

  3. After working through all of the questions posted throughout the lesson, you are encouraged to explore your own questions.

This lesson has a lot of information, but it is super useful in preparing you to complete the project. It is exciting to start putting your skills to practice, and we can't wait to see what you will create!
