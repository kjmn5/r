03. If/Else [Text]

If/Else

If/else logic is used in code and in the world all around us. Imagine we are pre-heating our oven to 425 degrees F. When the temperature of the oven hits 425, we want to know. However, if it isn't that hot yet, we will keep waiting.

Example

First, we might create a couple of variables:

oven.temp = NULL
is.it.ready.result1 = "It is ready!!!"
is.it.ready.result2 = "It isn't ready yet :("

I have stored NULL in our variables, as values that we will update later.

Now our if/else logic could be used in the following way:

if (oven.temp == 425){
    print(is.it.ready.result1)
} else {
    print(is.it.ready.result2)
}

You can see the first thing we will do is check if the oven.temp == 425. If it is, we print the first result, which I have created using 

is.it.ready.result1 = "It is ready!!!"

If the temp isn't 425, then I print the second result

is.it.ready.result2 = "It isn't ready yet :("

Testing Example 1 Code

We could test to make sure our if/else statement works correctly by updating our NULL place holder with some other value.

oven.temp = 385
is.it.ready.result1 = "It is ready!!!"
is.it.ready.result2 = "It isn't ready yet :("

if (oven.temp == 425){
    print(is.it.ready.result1)
} else {
    print(is.it.ready.result2)
}

This provides the output of 

"It isn't ready yet :("

If you change the oven.temp = 425, you will see the correct result as: 

"It is ready!!!"

Example 2

We could add to this example by adding logic for if the oven is too hot (if it is hotter than 425), not hot enough (if it is less than 425), or just right (if it is equal to 425).

Updating our code for this example we might change our results to the following three

is.it.ready.result1 = "oven is just right"
is.it.ready.result2 = "oven is too hot"
is.it.ready.result3 = "oven is not hot enough"

And then our logic might also be updated:

if (oven.temp == 425){
    print(is.it.ready.result1)
} else if (oven.temp > 425) {
    print(is.it.ready.result2)
} else {
    print(is.it.ready.result3)
}

Testing Example 2 Code

We could test to make sure our if/else statement works correctly by trying a few examples

oven.temp = 385
is.it.ready.result1 = "oven is just right"
is.it.ready.result2 = "oven is too hot"
is.it.ready.result3 = "oven is not hot enough"

if (oven.temp == 425){
    print(is.it.ready.result1)
} else if (oven.temp > 425) {
    print(is.it.ready.result2)
} else {
    print(is.it.ready.result3)
}

This provides the output of 

"oven is not hot enough"

If you change the oven.temp = 425, you will see the correct result as: 

"oven is just right"

Modular

Though not specifically related to if/else statements, understanding modular functionality is beneficial for the questions on the next page. The modular command has the syntax of %%, and it is used to identify the remainder when one number is divided by another number.

For example, the result of the below is zero, because 15 is exactly divisible by 3.

[Input]:  15 %% 3
[Output]: 0

However, the result of this is not zero, because 10 is not exactly divisible by 3

[Input]:  10 %% 3
[Output]: 1

It is common to check if a value is exactly divisible by another using %% along with ==.

[Input]:  10 %% 3 == 0
[Output]: FALSE
[Input]:  6 %% 3 == 0
[Output]: TRUE

This may come in handy for the second example on the next page!
