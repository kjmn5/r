05. Loops [Text]

Loops

One foundational topic of any programming language is the topic of loops. In R, you can write loops using the following syntax. Though spaces are not important, the parentheses (and curly braces } are important to have in the correct spots for your code to run.

for (val in 1:10){
    print(val)
}

This will print all values from 1 to 10. Notice again the parentheses and curly braces. All for loops will have this setup.

Example 1

Often when creating for loops, it is useful to create variables that will be placeholders that are updated throughout the loop.

If we want to sum all of the values from 2 to 7, we can change the above loop.

sum = 0
for (val in 2:7){
    sum = sum + val
}
print(sum)

This line here

sum = sum + val

takes the previous sum value on the right side, adds val, and then stores this as a sum to be used on the next iteration of the loop.

These iterations within the loop look like this:

sum + val
__________
  0 + 2
  2 + 3
  5 + 4
  9 + 5
  ...

Notice the result of the previous line is the new sum (the first value on the next line). The val is being iterated through as the second number in each line. This is illustrated in the image below.

Example 1

If instead of summing each of these values together, we wanted to store each sum in an array. We might create an empty vector that we can iterate through and store each sum. An example of this is shown in the code below.

sums = NULL
idx = 1
sum = 0

for (val in 2:7){
    sum = sum + val   # calculates new sum
    sums[idx] = sum    # stores sum in sums vector
    idx = idx + 1  # iterates to next index in the vector
}

Using sums = NULL means that we can define how sums works when we need to use it. This is a nice placeholder until we use it in our function. Then the idx value can point at a particular index in our vector which is where we store the next sum. 

idx = idx + 1

Since idx = 1 at the start of our code, the first sum value will go in the first index, then the above line changes this to the second index, and we put the second sum in that line. The result then is a vector of these sum values.

[Input]:  sums
[Output]: 2 5 9 14 20 27
