11. Passing And Creating Arguments [Text]

Passing And Creating Arguments in Functions

There are actually a few different ways we might pass arguments to a function. At a high level, these are

  1. By default

  2. By position

  3. By Name

Let's look at how this works in practice. Consider the function below, which is pretty close to the function you saw in the previous text.

happyGreet = function(greeting.type = "Hello", to.who = "Mohit"){
    result = paste(greeting.type, to.who, sep=" ")
    return(result)
}

You can see the function is happyGreet, which has two arguments: greeting.type and to.who.

Passing Arguments by Default

The first way that we can pass arguments is by default. In this case, the starting values in the function will be used, and no other values need to be provided to the function.

Here is an example using our happyGreet function.

[Input]:  happyGreet()
[Output]: "Hello Mohit"

By Position

You might also decide to pass argument values by position. Since the order of the arguments when we created the function is as shown here:

greeting.type = "Hello", to.who = "Mohit"

We can pass the arguments to our function in this order without having to say what the name of the argument is. 

[Input]:  happyGreet("Good Bye", "Jennifer")
[Output]: "Good Bye Jennifer"

We could also just change the greeting, but keep the default to.who value. However, we cannot do the opposite using "by position" arguments.

[Input]:  happyGreet("Good Bye") # this uses the default second argument
[Output]: "Good Bye Mohit"

By Name

The final, and most explicit, way to pass arguments to a function is by name. With this method, the order of the arguments doesn't matter and arguments not provided will use default values. Below are a few different ways you can pass an argument by name to a function.

[Input]:  happyGreet(greeting.type = "Buenos dias", to.who = "Kathy")
[Output]: "Buenos dias Kathy"

[Input]:  happyGreet(to.who = "Kathy", greeting.type = "Buenos dias")
[Output]: "Buenos dias Kathy"

[Input]:  happyGreet(to.who = "Kathy")
[Output]: "Hello Kathy"

[Input]:  happyGreet(greeting.type = "Buenos dias")
[Output]: "Buenos dias Mohit"

You can see the order doesn't matter, and you can leave out arguments that you want to use the default.
