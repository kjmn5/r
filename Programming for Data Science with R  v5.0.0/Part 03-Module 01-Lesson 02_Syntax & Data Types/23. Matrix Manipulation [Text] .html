23. Matrix Manipulation [Text]

In the next videos, you will be performing operations on matrices. Below is a refresher. However, if you are totally new to matrix operations, a full course is available at Khan Academy that you may want to consider working through to get comfortable with these and other matrix calculations.

Matrix Manipulation

There are a few matrix commands that are useful to understand when working in R. The first is that scalar operations are carried out across all of your matrix elements.

If you multiply a matrix by 10, the result is every element of the matrix multiplied by 10. Below you can see an example where I have created my.array as a 2x2 matrix.

[Input]:  my.array = array(seq(1,4,1), dim=c(2,2))
[Input]:  my.array
[Output]: 
          1    3
          2    4

We can then add 10 to all the elements:

Therefore the original values change from 

      1 -> 1 + 10          2 -> 2 + 10         3 -> 3 + 10       4 -> 4 + 10
[Input]:  my.array + 10
[Output]: 
          11    13
          12    14

We can do the same with multiplication, division, or subtraction.

Transpose

The transpose of a matrix flips the values in the rows to instead be in the columns. Below you can see an example of a starting matrix and the result after transposing it.

In R it is quite simple to transpose a matrix, as you use the t() function. Performing the operation in the image above can be done in the following way:

[Input]:  array1 = array(c(2, -10, 18, 4, 5, -7, -1, 11, 6), dim=c(3,3))
[Input]:  t(array1)
[Output]: 
          2    -10    18
          4      5    -7
         -1     11     6

Matrix Multiplication

You can also do matrix multiplication easily in R. A couple of reminders about matrix multiplication

  1. The number of columns in the first matrix must equal the number of columns in the second matrix.
  2. The value in the final matrix for any position is equal to the dot product of the row in the first matrix with the column in the second matrix. You can see this in the image below.

In order to complete the matrix multiplication shown in the image above in R, we perform use the operator %*%:

[Input]:  array1 = array(c(1, 3, 2, 4), dim=c(2,2))
[Input]:  array2 = array(c(5, 0, 6, 7), dim=c(2,2))
[Input]:  array1 %*% array2
[Output]: 
           5    20
          15    46

You can see that we first list all the elements in the first column followed by the elements in the second column. This pattern would continue no matter the number of rows or columns in our matrix.

cbind & rbind

The cbind and rbind commands are useful for adding columns or rows to existing matrices or creating new matrices! You can see an example in the image below.


As a reminder, you can pull a particular row of any matrice using square brackets. Consider the below, as this will be useful for a couple of the quizzes ahead!

We can create a matrix and pull the first row using [1, ]

[Input]:  array1 = array(c(1, -3, 0, 2), dim=c(2,2))
[Input]:  array1[1, ] # pulls the first row
[Output]: 
          1     0

We can pull say the third column of a matrix with [ ,3]

[Input]:  array1[ ,1] # pulls the first column
[Output]: 
          1     
         -3

We can bind the results either on columns

[Input]:  cbind(array1[ ,1], array1[1, ]) # 
[Output]: 
          1   1 
         -3   0

or the rows

[Input]:  rbind(array1[ ,1], array1[1, ]) # 
[Output]: 
          1  -3 
          1   0

Pretty cool, right!
