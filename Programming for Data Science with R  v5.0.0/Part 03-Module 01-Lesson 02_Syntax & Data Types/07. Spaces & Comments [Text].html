Spaces & Comments

The next page will specifically be on spaces and comments in R.

For most code you write **Spaces Do Nothing. **

4 + 5

and 

4 +          5

Both of these code blocks produce the result of 9.

Comments in R can be created with # . Comments are often useful to help others read through your code. They do not actually run any code that will produce a result. They are just meant to help guide people through your analysis. 

# The below will produce the result 9
# These comments will not be printed as a result
4 + 5 # This is a comment that won't hurt the result, 9 is still printed
