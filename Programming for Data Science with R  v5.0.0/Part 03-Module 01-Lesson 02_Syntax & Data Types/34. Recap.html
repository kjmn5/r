34. Recap

Recap

In this lesson, you learned a ton about working with data in R.

  1. You learned about data types including 
     a. numeric
     b. character
     c. logical
  1. You learned about vectors, and how we can create vectors using 
     a. vector()
     b. seq()
     c. c()
  1. You learned about comparison operators, and how you can use them on vectors and scalars. These included:
  a. > (greater than)
  b. < (less than)
  c. <= (less than or equal)
  d. >= (greater than or equal)
  e. == (equal)
  f. != (not equal)
  1. Then you learned how to subset arrays using indexing and square brackets. A few examples include:
  a. subset(array, array > 5) (only elements greater than 5) 
  b. array[c(1,3), 1:5] (first and third row, columns 1-5)
  c. array[1,2] (only the element in the first row, second column)
  1. You also learned about how to combine arguments:
  a.  a > 5 & b < 7 (and)
  b. a > 5 || b < 7 (or)
  1. You learned about a few matrix operations. These included:
  a. %*% (matrix multiplication)
  b. t(array) (transpose)
  c. *, +, - . / (scalar operations you can perform on an array)
  d. cbind(array1, array2) (combining columns into a single array)
  e. rbind(array1, array2) (combining rows into a single array)
  1. Finally, we looked at the iris dataset, which was a data.frame. You learned about additional operations on dataframes. A few of these operations include:
  a. head(df, 7) (show top 7 rows, tail works similarly)
  b. dim(df) (gives row, column of the dataframe)
  c. summary(df) (gives summary statistics on each column)
  d. min(df$col) (gives min of this column in the dataframe) 
  e. max(), sd(), mean(), median() (additional statistics you can calculate)

You now have a ton of operations you can use to work with data in R! Nice work!
